## Project Overview
A secure, real-time visitor management system for New Era University Library. 

### Key Features (Requirement Checklist):
- ✅ **Google OAuth Login:** Secure access via NEU Google accounts.
- ✅ **Role-Based Access Control:** Admin (`jcesperanza@neu.edu.ph`) vs. Regular User.
- ✅ **Real-time Statistics:** Dashboard cards showing Total Visitors, Students, and Staff.
- ✅ **Filtering Logic:** Admins can filter logs by clicking the Statistics Cards.
- ✅ **Automated Logs:** Captures Name, Email, College, and Reason for visit.

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: Supabase (Database & Auth)
